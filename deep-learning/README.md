# Deep Learning Module - MoldProject

Integrated professional AI development environment for MNIST classification.

## Structure
- `api/`: FastAPI inference service.
- `datasets/`: Training data (Git ignored).
- `docker/`: Container definitions.
- `outputs/`: Model checkpoints and training graphs.
- `training/`: Model architecture and training scripts.

## Quick Start

### Local Training
```bash
pip install -r requirements.txt
python training/train.py
```

### Docker Lab
```bash
docker-compose up --build
```
