## Title
Demonstrating the Feasibility of the VTT Model for Mold Risk Evaluation Across Building Contexts (PBL)

## Executive Summary
Based on Vereecken & Roels (2012), the VTT model, especially the updated VTT variant, is suitable as a cross-material screening tool for mold risk in PBL work. The model includes the main physical drivers of mold growth, has been extended beyond wood-based materials, and provides sensitivity classes for material mapping. The conclusion is not that VTT is perfect or universally exact, but that it is a defensible risk-evaluation model when its limitations are stated clearly and field verification is added.

# Word README - VTT Argument for PBL (Expanded Version)

## 1. Document Title
Demonstrating the Feasibility of the VTT Model for Mold Risk Evaluation Across Building Contexts (PBL)

## 2. Executive Summary
This document concludes that the VTT model is appropriate for PBL use across building contexts, especially when the updated VTT version is used. The reason is that the model includes the main physical factors of mold growth, has been extended to multiple building materials, and provides sensitivity classes for material mapping. The explanation in this document uses three PDFs: the main VTT paper as the theoretical base, one Indonesian-context paper, and one humid/air-conditioned building paper as supporting evidence.

That said, the academic claim must be written carefully: the paper does not prove absolute accuracy for every building condition. Therefore, VTT should be used as a risk-evaluation tool that still requires field validation.

## 7. Use Cases
### 7.1 Use Case 1 - Low-cost dwellings in Pasar Kliwon, Surakarta
This scenario is based on Murtyas et al. (2023). The paper reports field measurements in 17 low-cost dwellings in the Pasar Kliwon urban slum area of Surakarta, using indoor temperature and humidity time series to predict mold risk. The case is useful because it shows how VTT can be applied to real Indonesian housing conditions.

Key evidence:
- VTT applied to time-series temperature and humidity: P001 L037-P001 L039
- Daily RH above 80% lasted 2.2-12.3 hours: P001 L040-P001 L041
- Plywood walls showed higher cumulative mold growth risk: P005 L026-P005 L028; P006 L047-P006 L049

Implementation steps:
1. Install data loggers at representative points in each dwelling.
2. Use 10- to 60-minute intervals and collect data for at least 14-30 days.
3. Map the materials to sensitivity classes, for example plywood to sensitive or very sensitive and cement/brick to medium resistant.
4. Run VTT per element and prioritize interventions such as ventilation improvement or material upgrading.

### 7.2 Use Case 2 - Island residential buildings
This scenario is based on Zhang et al. (2020). The paper discusses marine/island housing with high temperature, high relative humidity, and saline exposure, and it gives material vulnerability rankings plus HVAC recommendations for mold prevention.

Key evidence:
- Material vulnerability ranking from reinforced concrete to wood: P001 L025-P001 L027
- Recommended indoor parameters for prevention: temperature 26-28°C and RH 50%-80% with dehumidification: P001 L028-P001 L031; P012 L031-P012 L032

Implementation steps:
1. Prioritize stronger envelope materials and avoid wood in highly exposed areas.
2. Set HVAC and dehumidification targets around 26-28°C and 50%-80% RH in occupied zones.
3. Validate the design using VTT or WUFI-based simulation before retrofit.
4. Use the result to choose between material substitution, HVAC adjustment, or surface protection.

## 8. Writing Guidance
1. Always attach the page-line anchors to every important claim.
2. Keep the conclusion careful: VTT is a risk-screening model, not an absolute prediction model.
3. State clearly when the argument is an adaptation of the PDF evidence rather than a direct claim from the PDF.

## 9. Presentation Flow
Use this order when explaining the model in a presentation or defense:
1. Define the problem: mold depends on temperature, humidity, time, and materials.
2. Explain why VTT is chosen: it includes the key variables and has an updated version.
3. Show the material evidence: four sensitivity classes and material mapping.
4. State the limitations: the model is not absolute and still requires validation.
5. Conclude: VTT is suitable as a PBL risk-evaluation model, not as a perfect predictor.
