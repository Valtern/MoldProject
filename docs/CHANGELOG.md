# Changelog

Semua perubahan penting pada proyek ini dicatat di file ini.

Format yang dipakai mengikuti pola sederhana: `Added`, `Changed`, `Fixed`, `Removed`, dan `Docs`.

## Unreleased

### Added
- Struktur dokumentasi lengkap di folder `docs/`.
- `docs/getting-started.md` untuk panduan instalasi dan menjalankan aplikasi.
- `docs/deploy.md` untuk panduan deploy ke environment target.
- `docs/architecture.md` untuk ringkasan arsitektur aplikasi.
- `docs/user-manual.md` untuk manual pengguna.
- `docs/CONTRIBUTING.md` untuk alur kontribusi dan PR.
- `docs/README.md` sebagai indeks dokumentasi.

### Changed
- Penyesuaian UI mobile agar tetap terpisah dari tampilan desktop.
- Perbaikan perilaku theme agar tidak berubah otomatis saat login.
- Penataan ulang dokumentasi agar mudah dibaca oleh developer dan pengguna akhir.

### Fixed
- Logout dialog mobile dibuat lebih kecil dan konsisten dengan tema.
- Tampilan device ID di mobile dibuat lebih rapi agar tidak menimbulkan horizontal scroll.

### Docs
- Tambahkan screenshot aktual ke folder `docs/assets/` setelah UI final disetujui.
- Tambahkan nomor versi rilis saat masuk ke produksi.
